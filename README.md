# 売上管理システム

売上管理システムは、スーパー向けの売上管理システムで、Spring BootとH2データベースを使用して売上データの管理、集計、閲覧が可能です。
本システムでは、売上データの登録、更新、削除ができ、日別・月別・商品別の売上集計やCSV形式でのデータ出力、簡易的なユーザー管理も可能です。

システムはJavaで開発され、フレームワークにはSpring Boot、データベースには組み込みのH2を使用しています。
ビルドにはMavenを使用し、フロントエンドはThymeleafで構築されています。
動作環境としてはJava 17以上、Maven 3.8以上、Eclipse 2023以上を推奨しており、
ブラウザはChrome、Edge、Firefoxで動作確認済みです。

セットアップは、まずリポジトリをクローンし、Eclipseで「Existing Maven Projects」としてインポートします。
その後、`./mvnw spring-boot:run` コマンドでアプリケーションを起動し、ブラウザで `http://localhost:8080` にアクセスすることでシステムを利用できます。

使い方は、トップ画面で売上一覧を確認し、「新規登録」から売上データを入力、既存データの編集・削除も可能です。
また、集計画面では期間別や商品別の売上を確認できます。データベースは組み込みのH2を使用しており、
コンソールURLは `http://localhost:8080/h2-console`、JDBC URLは `jdbc:h2:mem:testdb`、ユーザー名は `sa`、パスワードは設定されていません。

本システムはデモ用途のため認証や権限管理は簡易で、データはアプリ終了時に消える仕様になっています。
