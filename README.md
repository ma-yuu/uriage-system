# 売上管理システム

売上管理システムは、スーパー向けの売上管理システムで、
Spring BootとH2データベースを使用して売上データの登録、編集、削除、日別・月別・商品別の集計が可能です。
CSV形式でのデータ出力や簡易的なユーザー管理も行えます。

## 技術概要

- 言語：Java  
- フレームワーク：Spring Boot  
- データベース：H2（組み込みDB）  
- ビルドツール：Maven  
- フロントエンド：Thymeleaf  

## 動作環境

- Java 17以上  
- Maven 3.8以上  
- Eclipse 2023以上（推奨）  
- ブラウザ：Chrome / Edge / Firefox  

## セットアップと使用方法

システムを使うには、まずGitHub上のプロジェクトを自分のパソコンにコピーしてください。その後、Eclipseを開き、以下の手順でプロジェクトをセットアップします。

1. **プロジェクトのインポート**  
   Eclipseで「Existing Maven Projects」を選択し、プロジェクトをインポートします。  
   インポート時には、プロジェクト内にある `pom.xml` ファイルを指定してください。

2. **アプリケーションの起動**  
   インポートが完了したら、ターミナルやコマンドプロンプトで以下のコマンドを実行します。  
   ```bash
   ./mvnw spring-boot:run

3. **ブラウザでアクセス** 
   アプリケーションが起動したら、ブラウザで以下にアクセスしてください。
   http://localhost:8080

   トップ画面では登録済みの売上データの一覧を確認でき、「新規登録」ボタンから新しいデータを追加したり、既存のデータを編集・削除することも可能です。

本システムはデモ用のため、認証や権限管理は簡易で、アプリケーションを終了するとデータは消える仕様になっています。
