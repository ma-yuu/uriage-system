# 売上管理システム

売上管理システムは、スーパー向けの売上管理システムで、
Spring BootとH2データベースを使用して売上データの登録、編集、削除、日別・月別・商品別の集計が可能です。
CSV形式でのデータ出力や簡易的なユーザー管理も行えます。

## 技術概要

- 言語：Java  
- フレームワーク：Spring Boot  
- データベース：H2（組み込みDB）  
- ビルドツール：Maven  
- フロントエンド：Thymeleaf  

## 動作環境

- Java 17以上  
- Maven 3.8以上  
- Eclipse 2023以上（推奨）  
- ブラウザ：Chrome / Edge / Firefox  

## セットアップと使用方法

システムを利用するには、まずリポジトリをクローンしてローカル環境にコピーします。その後、Eclipseを開き、以下の手順でプロジェクトをセットアップしてください。

1. Eclipseで「Existing Maven Projects」を選択し、プロジェクトをインポートします。インポート時にはプロジェクト内の `pom.xml` ファイルを指定してください。  
2. インポートが完了したら、ターミナルまたはコマンドプロンプトで `./mvnw spring-boot:run` を実行し、アプリケーションを起動します。  
3. 起動後、ブラウザで `http://localhost:8080` にアクセスすると、売上管理システムのトップ画面が表示されます。  
4. トップ画面では登録済みの売上一覧を確認でき、「新規登録」ボタンから売上データを追加したり、既存データの編集・削除を行ったりできます。集計画面では期間別や商品別の売上を確認することも可能です。  
5. データベースは組み込みのH2を使用しており、ブラウザから `http://localhost:8080/h2-console` にアクセスすることで直接確認や操作が可能です。JDBC URLは `jdbc:h2:mem:testdb`、ユーザー名は `sa`、パスワードは設定されていません。  

本システムはデモ用途のため、認証や権限管理は簡易で、アプリケーションを終了するとデータは消える仕様になっています。
