# 売上管理システム

売上管理システムは、スーパー向けの売上管理システムで、
Spring BootとH2データベースを使用して売上データの登録、編集、削除、日別・月別・商品別の集計が可能です。
CSV形式でのデータ出力や簡易的なユーザー管理も行えます。

## 技術概要

- 言語：Java  
- フレームワーク：Spring Boot  
- データベース：H2（組み込みDB）  
- ビルドツール：Maven  
- フロントエンド：Thymeleaf  

## 動作環境

- Java 17以上  
- Maven 3.8以上  
- Eclipse 2023以上（推奨）  
- ブラウザ：Chrome / Edge / Firefox  

## セットアップと使用方法

まずリポジトリをクローンし、Eclipseで「Existing Maven Projects」としてプロジェクトをインポートします。
`pom.xml` を指定してインポートしてください。その後、ターミナルまたはコマンドプロンプトで `./mvnw spring-boot:run` を実行してアプリケーションを起動し、
ブラウザで `http://localhost:8080` にアクセスするとシステムを利用できます。
トップ画面では売上一覧を確認でき、「新規登録」から売上データを追加したり、既存データの編集・削除も可能です。
集計画面では期間別や商品別の売上を確認できます。データベースは組み込みのH2を使用しており、
コンソールURLは `http://localhost:8080/h2-console`、JDBC URLは `jdbc:h2:mem:testdb`、ユーザー名は `sa`、パスワードは設定されていません。
本システムはデモ用途のため認証や権限管理は簡易で、データはアプリ終了時に消える仕様になっています。
